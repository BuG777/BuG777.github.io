def spider_comment(page=0):
“””
爬取某页影评
:param page: 分页参数
:return:
“””
print(‘开始爬取第%d页’ % int(page))
start = int(page * 20)
comment_url = ‘https://movie.douban.com/subject/33717351/comments?start=0&limit=20&sort=new_score&status=P' % start
# 请求头
headers = {‘user-agent’: ‘Mozilla/5.0’}
try:
r = s.get(comment_url, headers=headers)
r.raise_for_status()
except:
print(‘第%d页爬取请求失败’ % page)
return 0
# 使用正则提取影评内容
comments = re.findall(‘(.*)‘, r.text)
if not comments:
return 0
# 写入文件
with open(COMMENTS_FILE_PATH, ‘a+’, encoding=r.encoding) as file:
file.writelines(‘\n’.join(comments))
return 1

C6 耐心|想象力决定生产力

058 易耗品与耐用品

每个人在面临四种比较基本的约束：东西不够，时间有限，互相依赖，需要协调。
欧文·费雪(Irving Fisher)在《利息理论》(The Theory of Internet,1930)一书中说“Investment is the balance of consumption over time .投资是时间维度上的平衡消费。”他认为，利息是人类社会最古老的现象之一，它起源于时间，起源于人们的耐心。
易耗品:一次性消费完毕的物品，如苹果
耐用品:耐用品在长时间的维度上体现价值，如苹果树，油画，学位，儿童，人力资本

059 不耐产生交易

耐心与自律，时局，文化背景有关
社会环境的不确定性会作用在利率上，以作为对推迟消费的补偿。

060 利率管制及其后果

在人类历史上大多数国家，收取利息被认为是一种不道德的行为，甚至被禁止。。
在美国，直到1980年颁布《储蓄机构放松管制和货币控制法》以后，才完全解除对利息的控制。
在中国，根据《最高人民法院关于审理民间借贷适用于法律若干问题的规定》,民间借贷不得超过年利率36%，超过部分不受法律保护。

在宗教和文学作品中都有对“利息”抵触的表现，比如在莎士比亚的《威尼斯商人》 中，向基督徒安东尼奥放贷的犹太人夏洛克经常被人们认为是一个唯利是图的小人，这与在文学作品中，很容易忽略对时间维度的感知。
这里存在一个固有印象，需要借钱的一般是急用钱的或低收入人群，放贷者借出钱，但收取利息，则会被认为是趁火打劫，且在过去医疗等水平低，不确定因素较多，民间利息较高，一旦还不清本金，会发生卖身等丧失自由的情况，因此导致某种仇富的情绪。
屏幕对面的你，是否认为只有工人、农民才是真正创造财富的人，而那些放贷者收利息是坐享其成？是否认为“等待、承担风险、推迟消费、提供流动性”不属于对社会的贡献？


油画《放贷者和他的妻子》 Quentin Matsys于1514年所作 现藏于卢浮宫

利率管制过当的情况下，有可能出现有钱人不愿意借钱，黑社会放贷，迂回支付利息等情况。

在某些地区，当地的政府会拨一种零利息小额贷款，如果审批不严格，会出现下面这种情况，钱被贷了，但是并不是由政策上希望用的人来使用，而是由有权钱有关系的人使用，那么制订之初的意愿就被违背了。

061利率变化支配资本现值

贴现率:未来价值(期值)转换为当前价值(现值)的打折程度。
贴现率影响个人自律程度:某些政商干部在早年没考虑到未来,只想到当下,造成一些”老寒腿问题”。
贴现率公式 pv现值 fv期值 r利率 n期数 pv = fv/(1+r)**n
